# Catálogo de Planetas — Proyecto React + Vite

Pequeña aplicación para el parcial: catálogo del sistema solar con integración 3D.

API principal: https://api.le-systeme-solaire.net/rest/bodies/

Dependencias principales:
- react
- react-dom
- react-router-dom
- vite (dev)

Comandos rápidos:

1. Instalar dependencias

```
npm install
```

2. Ejecutar en modo desarrollo

```
npm run dev
```

Estructura principal en `src/` con componentes, pages, services, hooks, data y utils.
